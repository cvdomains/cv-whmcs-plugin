# cv-whmcs-plugin
.cv domains whmcs plugin

A powerful WHMCS extension that lets domain registrars sell and manage this TLD seamlessly inside WHMCS. Our module plugs directly into your existing workflow, so your resellers can search, register, renew, transfer, and manage domains in real time — with live availability checks, automated pricing, and full lifecycle sync back to your registry.

Built for reliability and scale, it supports automated provisioning, EPP integration, custom pricing tiers, and detailed logging for every operation. You control policies and promotions; WHMCS handles the automation. No custom glue code, no manual workarounds — just a clean, native experience for your team and your resellers.

Give your partners a smooth, familiar WHMCS experience for your TLD, reduce support tickets, and speed up domain sales with a module that “just works.”

OVERVIEW:
The DOT CV Domain Reseller WHMCS Module provides domain registrars and domain resellers with a FREE domain registrar module, that brings about the automation of .CV domains provisioning and management into the WHMCS system. The module is secure, simple and easy to manage. Within 3 minutes you will be selling .CV domains.

The DOT CV (Cabo Verde ccTLD) Domain Reseller WHMCS Module supports domain registration for the following domains:

.cv - for anyone anywhere in the world. No restrictions.

MOSTLY FOR USE IN CABOVERDE
.com.cv - for Companies
.org.cv - for Not-for-Profit-Making Organisations or NGO’s
.gov.cv - for Government Entities (Requires Supporting Documents)
.edu.cv - for Institutions of Higher Education. (Requires Supporting Documents)

Instructions on how to install this module:

Download the file.
Upload and extract the module into the main WHMCS directory.
Go to Setup → Products/Services → Domain Registrars and activate module.
Set credentials for the API connection. (Send support@ola.cv email to request for API Key)
